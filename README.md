# Face-Liveliness-Detection-using-OpenCV

##Objective:-
To detect whether the detected face in a video stream is live (real human) or fake (photo/video/spoof) using computer vision and machine learning techniques.

##Basic Workflow:-
Start webcam feed.
Detect face region.
Detect facial landmarks.

##Analyze:
Eye blink patterns (EAR)
Mouth movement
Head motion

Combine features → make decision
If blinks, talks, or moves → Live
Else → Spoof
